# **App Name**: Hunar Yob

## Core Features:

- User Registration: Allow users to register with their name and other credentials, specifying whether they are an 'Usto' (artisan) or a 'Client'.
- Login/Logout: Enable users to log in and log out securely.
- Usto Profile Creation: For 'Ustos', allow them to create a profile showcasing their skills and upload up to 5 images of their work per listing.
- Listing Display: Display listings created by 'Ustos', showing their skills and uploaded images.
- Data Persistence: Store user registration data, profiles and listing details in LocalStorage.

## Style Guidelines:

- Primary color: A vibrant saffron (#FF7F50) to reflect the rich artistic traditions.
- Background color: Light beige (#F5F5DC), almost white, to keep focus on content.
- Accent color: Muted brown (#8B4513) for important user interface elements.
- Body and headline font: 'Alegreya', a humanist serif with an elegant feel.
- Note: currently only Google Fonts are supported.
- Use clean, simple icons related to user roles and artisan categories.
- A clean and intuitive layout to ensure ease of use for both 'Ustos' and 'Clients'.
