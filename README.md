# Ҳунар Ёб — Платформаи устоҳо

Ин барнома бо истифода аз NextJS ва Firebase сохта шудааст. Ин дастур ба шумо кумак мекунад, ки кодро ба GitHub гузоред ва сомонаро нашр кунед.

## Қадами 1: Гузоштани код ба GitHub

Барои он ки коди шумо дар GitHub бошад, ин қадамҳоро иҷро кунед:

1. **Сохтани Репозиторий**:
   - Ба [github.com](https://github.com) ворид шавед ва тугмаи **"New repository"**-ро пахш кунед.
   - Номи лоиҳаро нависед (масалан: `hunar-yob`) ва **"Create repository"**-ро пахш кунед.

2. **Иҷрои фармонҳо дар Терминал**:
   Дар папкаи лоиҳа терминали худро кушоед ва фармонҳои зеринро пай дар пай нависед:

   ```bash
   # Инициализатсияи Git
   git init

   # Илова кардани ҳамаи файлҳо
   git add .

   # Сабти аввалини код
   git commit -m "Аввалин боргузорӣ"

   # Пайваст кардани GitHub (URL-и худро аз GitHub нусхабардорӣ кунед)
   git remote add origin https://github.com/ИМЗОИ_ШУМО/hunar-yob.git

   # Гузоштани код ба GitHub
   git branch -M main
   git push -u origin main
   ```

## Қадами 2: Нашр дар Firebase (Publicate)

Пас аз он ки код дар GitHub пайдо шуд:

1. Ба [Firebase Console](https://console.firebase.google.com/) гузаред.
2. Лоиҳаи худро интихоб кунед.
3. Дар менюи чап **App Hosting**-ро интихоб кунед.
4. Тугмаи **"Get Started"**-ро пахш кунед ва репозиторийи GitHub-и худро интихоб кунед.
5. Firebase автоматӣ сомонаро месозад ва ба шумо URL медиҳад.

## Амният ва Маблағҳо
- **Firebase Firestore**: Ҳамаи маълумотҳо дар сервери воқеии Google сабт мешаванд.
- **Идентификатсия**: Корбарон бояд шиносномаи худро бор кунанд, то ҳамён фаъол шавад.
- **Пардохтҳо**: Барои пардохтҳои ҳақиқӣ бо "Корти Миллӣ", шумо бояд API-и яке аз бонкҳои Тоҷикистонро (Алиф, Эсхата ва ғ.) пайваст кунед.
