
# Ҳунар Ёб — Платформаи устоҳо

Ин барнома бо истифода аз **NextJS** ва **Firebase** сохта шудааст. Ҳамаи маълумотҳо дар базаи маълумоти воқеии Google (Firestore) сабт мешаванд.

## Қадами 1: Гузоштани код ба GitHub

Барои он ки коди шумо дар GitHub бошад, ин қадамҳоро иҷро кунед:

1. **Сохтани Репозиторий**:
   - Ба [github.com](https://github.com) ворид шавед ва тугмаи **"New repository"**-ро пахш кунед.
   - Номи лоиҳаро нависед: `hunar-yob`.
   - **"Create repository"**-ро пахш кунед.

2. **Иҷрои фармонҳо дар Терминал**:
   Дар папкаи лоиҳа терминали худро кушоед ва фармонҳои зеринро нусхабардорӣ карда, ворид кунед:

   ```bash
   # Инициализатсияи Git
   git init

   # Илова кардани ҳамаи файлҳо
   git add .

   # Сабти аввалини код
   git commit -m "Нашри аввалин бо Firebase"

   # Пайваст кардани GitHub (URL-и худро аз саҳифаи GitHub-и сохтаатон нусхабардорӣ кунед)
   git remote add origin https://github.com/ИМЗОИ_ШУМО/hunar-yob.git

   # Гузоштани код ба GitHub
   git branch -M main
   git push -u origin main
   ```

## Қадами 2: Нашр дар Firebase (Online)

Пас аз он ки код дар GitHub пайдо шуд:

1. Ба [Firebase Console](https://console.firebase.google.com/) гузаред.
2. Лоиҳаи худро интихоб кунед.
3. Дар менюи чап **App Hosting**-ро интихоб кунед.
4. Тугмаи **"Get Started"**-ро пахш кунед.
5. Репозиторийи GitHub-и худро (`hunar-yob`) интихоб кунед ва тугмаи "Next"-ро пахш кунед.

### Ҳалли мушкилоти "Project creation quota exceeded"
Агар ҳангоми сохтани лоиҳа хатогии квота барояд:
- Ба лоиҳаҳои кӯҳнаи худ дар Firebase Console равед.
- Лоиҳаҳои нолозимро нест кунед (Project Settings -> Delete Project).
- Ё як лоиҳаи мавҷударо барои ин барнома истифода баред.

## Амният ва Маблағҳо
- **Firebase Firestore**: Маълумоти воқеӣ ва "100% онлайн".
- **Идентификатсия**: Корбарон бояд шиносномаи худро бор кунанд, то ҳамён фаъол шавад.
- **Пардохтҳо**: Барои пардохтҳои ҳақиқӣ бо "Корти Миллӣ", шумо метавонед API-и яке аз бонкҳои Тоҷикистонро дар оянда илова кунед.
