
# KORYOB 2 TJ — Платформаи устоҳо (Версияи 1.0.0 — 2026)

## 🚀 Чӣ тавр кодро ба GitHub навсозӣ кунем? (Update Code)

Агар дар терминал ҳарфҳои тоҷикӣ сурх шаванд, ин фармонҳои соддаро истифода баред:

1. **Ҳамаи файлҳоро омода кунед:**
   ```bash
   git add .
   ```

2. **Тағйиротро сабт кунед (Commit):**
   ```bash
   git commit -m "Update 2026 version"
   ```

3. **Ба GitHub фиристед (Push):**
   ```bash
   git push origin main --force
   ```
