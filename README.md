# Ҳунар Ёб — Платформаи устоҳо

Ин барнома бо истифода аз NextJS ва Firebase сохта шудааст.

## Чӣ тавр барномаро нашр (Publish) кунем?

Барои он ки сомонаи шумо дар интернет бо домени худ (масалан `hunaryob.tj`) дастрас шавад, ин қадамҳоро иҷро кунед:

1. **GitHub**:
   - Дар [github.com](https://github.com) акаунт созед.
   - Репозиторийи нав (New Repository) созед ва коди ин барномаро ба он ҷо бор кунед (Push).

2. **Firebase Console**:
   - Ба [Firebase Console](https://console.firebase.google.com/) ворид шавед.
   - Лоиҳаи (Project) худро интихоб кунед.
   - Дар менюи чап ба бахши **App Hosting** гузаред.

3. **Пайвастшавӣ**:
   - Тугмаи "Get Started"-ро пахш кунед.
   - Репозиторийи GitHub-и худро интихоб кунед.
   - Танзимоти автоматиро қабул кунед.
   - Firebase сомонаро автоматӣ build мекунад ва ба шумо як URL-и ройгон медиҳад.

4. **Домени шахсӣ**:
   - Дар бахши App Hosting шумо метавонед домени шахсии худро (`.tj` ё дигар) илова кунед.

## Амният ва Маблағҳо
- Барнома бо Firebase Firestore пайваст шудааст, ки маълумоти корбаронро 100% онлайн ва бехатар нигоҳ медорад.
- Низоми "Шартномаи бехатар" (Escrow) маблағҳоро дар дохили базаи маълумот "заблок" мекунад, то иҷрои пурраи кор.
- **Огоҳӣ**: Барои пардохтҳои ҳақиқии бонкӣ (Korti Milli), шумо бояд бо яке аз бонкҳои Тоҷикистон (масалан Алиф ё Эсхата) шартнома бандед ва API-и онҳоро пайваст кунед.
